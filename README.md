## Installation

### From Chrome Web Store
1. Visit the [Matcha Chrome Extension](https://chrome.google.com/webstore/detail/matcha) page
2. Click "Add to Chrome"
3. Sign in with your Google account
4. Upload your CV and start matching!

### Manual Installation (Developer Mode)
1. Download the latest release ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

## How to Use

### Getting Started
1. **Sign in** with your Google account
2. **Upload your CV** (PDF or TXT format)
3. Navigate to any job posting

### Analyzing Jobs
1. Look for the **Matcha floating button** in the bottom-right corner
2. Hover over it and click **"Analyze Job"**
3. View your match score, gap analysis, and tailored bullets
4. Click **"Copy Bullets"** to paste into your application

### Tracking Applications
1. Click **"Add to Tracker"** to save a job
2. Open the tracker panel to view all saved jobs
3. Drag jobs between columns to update status
4. Click on a job to add notes or revisit the posting

## FAQ

**Q: Which file formats are supported for CV upload?**
A: PDF and TXT files are currently supported.

**Q: Does it work on all job sites?**
A: Yes! Matcha works on any webpage with job description content.

**Q: Is my CV data sent anywhere?**
A: Your CV is stored locally. It's only sent to our secure servers during analysis, and is not stored permanently.

**Q: Can I use it without signing in?**
